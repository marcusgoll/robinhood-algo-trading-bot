# Responsive Testing

**Purpose**: Test UI on mobile, tablet, and desktop sizes.

---

## Device Sizes

**Mobile**: 375x667 (iPhone SE)
**Tablet**: 768x1024 (iPad)
**Desktop**: 1920x1080

**Using Chrome DevTools**:
```
1. Open DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Select device preset or custom size
4. Test layout at each size
```
